香港孩子對本地公共遊樂場的公式最熟悉不過了，單單是滑梯，質感顏色高度斜度和滑度，閉上眼都能認出。它們不斷複製相同的遊戲經驗，除了方便管理、低成本、極低的風險系數外，還揭示了社會對孩子的什麼想像？抑或零想像？

《端傳媒》採訪中外遊樂場專家、園境設計師和政府部門，當然還有遊樂場的主角──孩子，嘗試把小朋友的需要重置到遊樂場設計中，開啟「我要真遊樂場」的討論。

畢竟，遊樂場上怎能沒孩子？

執藥式公園、倒模式童年

View Image

如果七、八歲的孩子告訴你「遊樂場很無聊」，然後埋首手機遊戲，請不要驚訝，這正正是大人有份種下的因。

除了極少數例外，香港的公共遊樂場一律有一樣的鞦韆（只能前後擺動的單人鞦韆）、一樣的滑梯（高一點點都被拆下了）、一樣的攀爬組合（哪個位置裝梯子哪兒有柱子，孩子都猜得中）、一樣的塑料設施和一樣的保護地墊。那些已經幻化成「常餐」、「快餐」和「特餐」標準化元素，教人忍不住搬出麥兜語錄──「嚟嚟去去有啲肉有啲菜撈埋撈埋打個生粉芡上去嗰啲嘢咁囉」（來來去去就是些肉啊菜啊混來混去放點芡粉這些東西咯）。

遊樂環境顧問袁漢昌叫這些做「執藥式（意即千遍一律）」遊樂場，那些「藥」來自供應商重重複複的組件。「這裏執幾件，那裏執幾件，看來很多，其實一樣。」他說：「但遊樂場不該只是執藥。」

View Image

袁漢昌：為孩子的遊樂權益發聲

他服務的智樂兒童遊樂場協會是爭取兒童遊戲權利的非政府機構，認為遊戲是成長必需，所以遊樂場的價值從不止於啟發學習，更不該只是給孩子「放電」而已。

1.1遊樂場的主角（小朋友）在哪裏？

遊戲是兒童權利──這句話聽來理所當然，但在很多地方其實是要捍衛的，比如香港。袁漢昌感受最深的經驗，來自他曾經參與的政府工程項目會議。

「與會者（作為用戶及管理者的康文署和房屋署，設計者的建築署，及園境設計師等）會討論用色和結構好不好看？與周邊環境是否配合？裝置是否安全？完全略去我們最該問的：這個社區的孩子需要怎樣的遊樂場？這樣設計孩子會否覺得好玩？」

當安全第一、減少投訴、設施耐用可替換等，成為主要考慮因素，點選現成組件便最省事，甚至已成慣性。「稍有不同的提議很輕易就被擱下，哪怕只有一人提出質疑。」袁漢昌說。

1.2從沙池看管理

譬如，沙池。在遊樂環境顧問眼中，沙池有很高的遊樂價值。沙本身是有趣的感官體驗，可塑性高，加點水便有不同質感，大小朋友都能從中創作和耍樂。所以袁漢昌一有機會便建議沙池，但得到的回應令人沮喪──

可以給孩子一個沙池嗎？

某次與會者還說，香港沙灘多，所以公園不需要沙池。「他可能不知道，基層家長帶孩子去一趟沙灘多不容易。」袁漢昌說。

《端傳媒》去信康樂及文化事務署查詢，對方回覆：「本署對於在兒童遊樂場提供水及沙等遊戲設施持開放態度，並會積極研究將不同的元素融入於兒童遊樂設施之中。」

現時康文署轄下約有七百個戶外兒童遊樂場，設有沙池的只有香港公園一個，至於哪個公園能玩水？官方回覆是「在石澳泳灘有一套遊戲組合設於沙灘之上。」

建築署高級建築師陳志威則指出，該署按用戶部門要求提供設施，過程中會了解後者在運作和管理方面的意見，草擬初步方案後諮詢區議會。

他說遊樂設施必須符合國際標準，不能為刺激而犧牲安全。但同樣遵照國際標準的海外遊樂場，為何又能保留遊樂經驗的多樣性？陳志威否認本地遊樂經驗單調，但就指出署方確實受限於供應商的提供，至於可否主動要求不同款式又或者度身訂造，他的回覆是「建築署不會開發設施，亦不是（遊樂場）項目的倡議者」。

那麼，設置沙池又難在哪裏？陳志威答：「香港很多地方都有沙池，譬如沙田和屯門。」但它們都拆除多年了。

1.3當限制成為理所當然

「其實業界也有有心人，但勢孤力弱。」袁漢昌說：「我們要做的，是一起為小朋友發聲。」

他口中其中一位有心人，是香港園境設計師學會副會長、香港高等科技教育學院設計學院助理教授陳元敬，我在一個發布設計學院學生得獎消息的記者會上跟他見面。那是「共融遊樂場概念設計比賽」，香港第一個針對遊樂場設計的比賽。

View Image

陳元敬：公共空間限制多多，最後變成為管理者設計，而不是市民用家。

會上，得獎學生分享了很多有趣意念，但被問及設計中為何沒鞦韆時，學生的回應竟然是「安全考慮」。

後來我跟陳元敬提起，他說：「有些想法確實入了腦──學生覺得危險，繼而失去反思，但設計師最重要的是反思，反思自己創造出來的空間其實意義何在？」

View Image
View Image

「共融遊樂場概念設計比賽」很多得獎作品都展現了令人興奮的設計意念，但香港小朋友暫時只能望梅止渴。

「今日的遊樂場高度講究安全，但帶挑戰性的遊樂設施，也是孩子發掘自我能力，學習風險評估的重要一環，是小朋友成長的一部分，不能拿走。」

「對於公共空間，年輕人在香港從沒見過多少好東西，這很糟糕。我們這輩尚可回憶小時候在公園自由自在踏單車，但新一代成長時都沒有這些經驗，周遭限制多多，反而覺得那些限制理所當然。」

1.4一個投訴滅絕一個物種

可曾把這些思維帶進政府會議？「那是對牛彈琴，他們只會告訴你：會有投訴。」

陳元敬說了虎尾蘭的故事。園境設計師為公共空間挑選植物有很多制肘：落葉樹不要、果樹不要、長得太快的不要、打理工夫多的不要⋯⋯某次他們建議虎尾蘭。那是一種耐陰耐旱，活力很強的常綠草本植物。

但官員說，本區不歡迎虎尾蘭，因為曾經有一個家長，向區議會投訴葉子割腳。

View Image

有問過小朋友想要怎樣的遊樂場嗎？康文署和建築署回覆，他們問過區議會。

「這正是我們社會的縮影，公園為何那麼多禁止標示？公共空間為何愈來愈多東西不能做？海濱為何不准放風箏？滑梯為什麼只有一個樣子？甚至沒地方玩沙玩水？其實都是同一道理。可能因為某時某人某個投訴，有些東西便告消失。大家是不是反應過敏了？」

「那些是為管理者而做的設計，不是為小朋友。」陳元敬說。

「在現制度下，管理者有最大的話語權，最終可以拒收設施。要是他們不同意，一個新設施根本不能出現。」

他引私人物業的設計流程來做對比：首先設計師為發展商規劃吸引用家的藍圖，建成後交予物業管理公司管理。「倘若在設計階段必須事事問准管理公司，很多設計根本不可能發生。」

1.5新加坡經驗：家長有責任

香港的遊樂場愈見單一，反觀鄰近新加坡的遊樂空間益發多彩，尤其是玩水設施。陳元敬曾經跟當地同業討論這個趨勢，「他們說，新加坡也經歷了一個過程。從前講求低風險，但近年銳意教育市民：你有責任保障自己孩子的安全，有權決定使用不使用，政府不會因而不提供某些遊樂設施。」

本地的管理者不是沒見過新穎遊樂場設計。在那些沒有孩子聲音的會議上，間中有人分享「我在新加坡/日本/美國見過，我帶孩子玩過」，但要是放在本地遊樂場的話，他們會說：「香港家長一定不能﹗」

袁漢昌說：「他們覺得香港家長一定會投訴。但我覺得這種投射，跟現代家長對遊樂場的理解，是有差異的。」

1.6香港公園事件：還我長滑梯

今年四月，香港公園四米高的長滑梯因為零件損毀而拆卸，有家長動員網上聯署，要求原址重置──因為那已是香港公共遊樂場裏碩果僅存的長滑梯了。風頭火勢，康文署邀袁漢昌提供意見，劈頭便問：「為什麼家長那樣緊張這條長滑梯？」

「我答，你們都拆了很多，只是之前沒多少人知道罷，其實你們有沒有留意，香港家長對公共空間的要求提高了？」袁漢昌說。

View Image

曾經讓很多孩子樂透的香港公園長滑梯，已成回憶。

他指出，新設施使用數年後折舊，往往便翻出另一頁「遊樂場血淚史」，因為管理者有權決定維修或拆卸。「所以管理者很需要遊戲教育，因為他們要幫忙維護遊樂場的遊戲價值，否則再精彩的東西，幾年後亦只會變回『執藥』式組件。」

「而家長也要懂得發聲，對好的遊樂設施正面評價，讓業界的有心人得到支持，夠膽為小朋友爭取。要是見到好的你不說出來，管理者又只聽到家長投訴這個不安全那個髒，那末我們只能回歸那些單一不好玩的遊樂場『套餐』。」

怎樣的遊樂場，有怎樣的孩子

Susan Goltsman

Susan Goltsman任教於美國加州大學柏克萊分校，專業範疇是遊樂空間。去年她來港主持共融遊樂場工作坊時，我逮了個機會問她如何評價本地遊樂場？「它們全都一樣，像直接從商品目錄中掏出來，有點悶，跟周遭的環境沒有很強的聯繫……」她輕笑一聲，續說﹕「我這樣說只是為了顯得有禮貌。」

這回，《端傳媒》越洋訪問了Susan Goltsman。

Q1端：你認為怎樣才是一個好的遊樂場？

Susan：我們必須知道，小孩子遊戲是正事，不是浪費時間。而戶外遊樂設施位於複雜多元的環境中，又較其他場所更有發掘空間。

好的遊樂場該包含一系列體驗元素，包括刺激感官發展、社交互動、提供想像空間，和融入周遭的自然環境等。孩子可以扮演各種角色，也可以主導自己的遊戲，絕對不只是一組爬上滑下的遊樂設施。空間再小，也可以豐富。

Q2端：你如何看安全這元素？

Susan：我們在設計上有很多安全標凖，但依循後，遊樂場依然可以刺激好玩。

必須明白，挑戰(challenge)和危害(hazard)是兩回事。危害是一口釘子露了出來，你不留意會被割傷。挑戰是讓孩子爬得很高，但同時提供足夠保護，讓他們決定是否嘗試。每個人都要學習面對風險，這也是遊樂場上的學習。

保養遊樂場對安全很重要，但如果你過分關注這一環，最後只會得到一式一樣的塑膠設施，因為它們太好保養了。有些人花很大力氣來保養哥爾夫球場，是因為明白它的價值；而遊樂場對人類發展而言，價值要比哥爾夫球場高得多。

Q3端：不同文化對安全二字，有很不同的演繹吧？

Susan：這絕對因文化和環境而異。幾年前我在巴勒斯坦參與興建遊樂場，發現防止兒童從遊樂場跑到街上很重要，否則孩子可能會吃子彈。至於在香港和中國，我發現人們特別不喜歡孩子弄髒。

Q4端：你說香港家長不喜歡髒，可有親身體會？

Susan：某年我在海洋公園看到一個大型噴水池，在美英等地那種地方定必好玩透頂，孩子會跑進去弄濕自己。但人們告訴我，香港人才不會那樣。我在香港幾乎找不到能挖沙和玩水的遊樂場，沒有大自然元素。這樣確實不會弄髒，但孩子本來就該玩得髒兮兮，現有的遊樂環境顯然不足。

View Image

香港公園是康文署轄下唯一設沙池的公共遊樂場。

Q5端：你對香港的遊樂場管理有何體會？

Susan：太多管理、太多規則。譬如，大人為何認為滑梯只能滑下，不能爬？

管理者有很多擔心，但其實只要重視遊戲，一定找到解決方法。管理者是要服務人們，不是倒過來限制孩子的需要。

Q6端：如果說遊樂場反映一個社會如何看孩子，你從香港的遊樂場看到什麼？

Susan：毫無疑問，遊樂場告訴你一個社會的價值系統，就像一個課室如果有點亂，到處都是小朋友的創作，很多活動在進行，那麼裏頭必定有一位以孩子為中心的老師，關心孩子真正的學習和自我表達。如果，你走進的課室完美地整潔，老師在講台上說書，那是很單向的教育方式。

香港的遊樂場，反映了政府想把孩子都留在指定地方玩，為了自己方便，所以到處放相同的東西，他們關心維修多於真正的遊戲。

有怎樣的遊樂場，就有怎樣的孩子，所以更要好好投資。地球上有些地方的人只想要麵包，有塑膠遊樂設施已經夠好，但你們老早就該超越這階段。

香港有不一樣的遊樂場嗎？

看了世界各地的好東西，難免要問，香港究竟有沒有？

據說，還有一些。

譬如觀塘海濱公園遊樂場有齊備的感官遊戲，包括香港地買少見少的氹氹轉。西九龍圓方的天台遊樂場引入歐洲式設施，提升挑戰性，也開闊玩法的想像。將軍澳寶康公園遊樂場集合多元化元素，滿足不同能力的小朋友。油塘大本型天台遊樂場試圖打破「設施才是遊戲」的框框，是讓遊戲融入環境的新嘗試。

我們綜合專家和網上家長的心水推介，挑出幾個「滄海遺珠」帶孩子試玩，也沿路聆聽孩子對好玩遊樂場的訴求。

說到底：他們才是真正用家。
